# 🌍 Currency Converter using Fetch API

A responsive Currency Converter built using **HTML**, **CSS**, and **Vanilla JavaScript**. It fetches live exchange rates from a free Currency API and allows users to convert between multiple international currencies with dynamically updated country flags.

---

## 🚀 Live Features

- 🌐 Real-time currency conversion using Fetch API
- 💱 Supports multiple international currencies
- 🚩 Dynamic country flags based on selected currency
- 🔄 Swap currencies with a single click
- ✅ Input validation for invalid amounts
- 📱 Responsive and clean user interface
- ⚡ Fast and lightweight (No external JavaScript frameworks)

---

## ⚙️ How It Works

1. User enters the amount.
2. Selects the source currency.
3. Selects the destination currency.
4. The application fetches live exchange rates using Fetch API.
5. Calculates the converted amount.
6. Displays the result instantly.
7. Updates country flags dynamically based on selected currencies.
